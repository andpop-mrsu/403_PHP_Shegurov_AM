# Task02: Веб-приложение «Арифметическая прогрессия»

Веб-версия игры из спецификации, реализованная на чистом PHP с использованием базы данных SQLite.

## Описание проекта

Приложение представляет собой веб-версию игры **«Арифметическая прогрессия»**.

Игроку показывается арифметическая прогрессия из 10 чисел со случайным шагом.  
Одно из чисел скрывается и заменяется символами `..`.  
Задача игрока — определить пропущенное число и ввести ответ в форму.

После отправки ответа приложение:

- проверяет правильность ответа;
- выводит результат игры;
- при ошибке показывает правильный ответ и полную прогрессию;
- сохраняет сведения об игре в базу данных SQLite;
- позволяет просматривать историю всех сыгранных партий на отдельной странице.

## Основные возможности

Приложение:

- генерирует арифметическую прогрессию из 10 чисел;
- случайным образом скрывает один элемент прогрессии;
- принимает ответ игрока через веб-форму;
- отображает результат проверки ответа;
- сохраняет данные о каждой игре в базу данных;
- показывает историю игр на отдельной странице.

## Какие данные сохраняются в базе данных

Для каждой сыгранной партии сохраняются:

- имя игрока;
- дата и время игры;
- результат игры;
- прогрессия с пропущенным элементом;
- полная прогрессия;
- пропущенное число;
- ответ игрока.

## Запуск

1. Перейти в каталог задания:

```bash
cd Task02
```

2. Запустить встроенный PHP-сервер:

```bash
php -S localhost:3000 -t public
```

3. Открыть в браузере:

- `http://localhost:3000/` — игра
- `http://localhost:3000/history.php` — история игр
