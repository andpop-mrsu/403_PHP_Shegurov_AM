# Arithmetic Progression

Консольная игра на PHP, в которой игроку нужно определить пропущенное число в арифметической прогрессии.

## Описание

Игроку показывается ряд из 10 чисел, образующий арифметическую прогрессию со случайным шагом.  
Одно из чисел заменено на `..`.  
Задача игрока — определить, какое число пропущено.

После завершения игры выводится результат:
- при правильном ответе — сообщение об успехе;
- при ошибке — сообщение с правильным ответом и полной прогрессией.


## Локальный запуск

```bash
cd Task01/progression
composer install
php bin/progression
```
