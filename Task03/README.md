# Task03: Арифметическая прогрессия - SPA + REST API на Slim

В данной работе игра «Арифметическая прогрессия» реализована в виде Single Page Application.

## Используемые технологии

- **Frontend:** `public/index.html`, `public/app.js`, `public/styles.css`
- **Backend:** `public/index.php` на PHP с использованием Slim
- **База данных:** SQLite (`db/progression.sqlite`)

## Суть игры

Игроку показывается арифметическая прогрессия из 10 чисел, в которой одно число заменено на `..`.

Задача игрока - определить пропущенное число.

После ответа:

- если число угадано правильно, выводится сообщение об успешном результате;
- если ответ неверный, показывается правильное число и полная прогрессия.

Информация о всех играх и ходах сохраняется в базе данных SQLite.

## Реализованные REST API маршруты

- `GET /games` - получить список всех игр из базы данных;
- `GET /games/{id}` - получить информацию о конкретной игре и всех ее ходах;
- `POST /games` - создать новую игру и сохранить ее в базе данных;
- `POST /step/{id}` - сохранить ход для игры с указанным идентификатором.

Также для корректной работы через встроенный сервер PHP поддерживаются маршруты через `index.php`:

- `GET /index.php/games`
- `GET /index.php/games/{id}`
- `POST /index.php/games`
- `POST /index.php/step/{id}`

## Что сохраняется в базе данных

Для каждой игры сохраняются:

- имя игрока;
- дата и время начала игры;
- дата и время завершения игры;
- статус игры;
- результат игры;
- прогрессия с пропуском;
- полная прогрессия;
- пропущенное число.

Для каждого хода сохраняются:

- идентификатор игры;
- номер шага;
- дата и время ответа;
- прогрессия с пропуском;
- полная прогрессия;
- правильный ответ;
- ответ игрока;
- признак правильности ответа.

## Установка и запуск

1. Перейти в каталог задания:

```bash
cd Task03
```

2. Установить зависимости (Slim):

```bash
composer install
```

3. Запустить встроенный сервер PHP:

```bash
php -S localhost:3000 -t public
```

4. Открыть SPA:

- `http://localhost:3000/`
